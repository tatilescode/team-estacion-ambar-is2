# Estación Ámbar 

Este repositorio contiene la documentación, diagramas, prototipos y gestión de sprints del proyecto **Estación Ámbar**, desarrollado para el curso **TS359 — Ingeniería de Software II** en la **Universidad del Valle de Guatemala, Campus Altiplano**.

---

##  Descripción del Proyecto

**Estación Ámbar** es un videojuego de terror y supervivencia en primera persona (*First-Person Survival Horror*). Inspirado en dinámicas de evasión y movimiento modular por zonas, el juego pone al jugador en los zapatos de una técnica de mantenimiento que debe reparar los sistemas de una instalación de entretenimiento infantil abandonada mientras evade autómatas defectuosos que reaccionan al ruido y la luz.

## Herramientas y Tecnologías Utilizadas

Para el desarrollo de este proyecto, el equipo utilizará el siguiente ecosistema de herramientas:

*   **Motor de Videojuegos:** Unity o Godot Engine (Gratuitos, ideales para desarrollo ágil en 3D).
*   **Control de Versiones:** GitHub (Repositorio público para control de cambios).
*   **Gestión de Tareas:** Notion / Jira (Tableros Kanban para seguimiento de entregables).
*   **Diseño de Interfaces:** Figma (Diseño de menús, wireframes y flujo de pantallas).
*   **Modelado y Assets:** Blender / Unity Asset Store (Recursos libres de uso).
